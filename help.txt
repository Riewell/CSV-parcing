  Значения параметров запуска имеют приоритет над значениями из файла настроек.
  Параметры указываются по отдельности, разделяются пробелами
  (то же относится и к значениям параметров).
  В случае задания нескольких параметров будет учтён только первый
  указанный параметр/первое значение.

  Параметры:
  
  [-o ФАЙЛ|--open ФАЙЛ]		Открыть указанный файл для
  				его обработки (вычисления сумм значений
  				в указанном столбце в разрезе товарных
  				групп/подгрупп) и записи вычисленных
  				значений во временный файл
  				(с расширением "tdat")
  
  [-a ФАЙЛЫ|--avarage ФАЙЛЫ]	Открыть указанные временные файлы
				*.tdat для их обработки (нахождения
				среденей величины вычисленных ранее значений
				по товарным группам) и записи
				вычисленных результатов во временный файл
  				(с расширением "adat")
  
  [-c ФАЙЛЫ|--combine ФАЙЛЫ]	Открыть указанные временные файлы
  				*.adat для извлечения вычисленных
  				ранее значений и их последовательной
  				записи в результирующий файл "result-*.csv"
  				(для возможности дальнейшей обработки
  				сторонними программами,
  				в т. ч. табличными процессорами)

  [-h]				Показать краткую справку
				по использованию и выйти.

  [--help]			Показать полную справку с описанием
				возможных параметров и выйти.
				
  Коды возврата:
  0 - обработка успешно завершена
  1 - общая некритичная ошибка (напр.: не указан параметр;
      в обрабатываемом файле есть неподдерживаемое форматирование и т. д.)
  2 - ошибка памяти во время выполнения
  3 - критичная ошибка (например, невозможность записи файлов на диск)
